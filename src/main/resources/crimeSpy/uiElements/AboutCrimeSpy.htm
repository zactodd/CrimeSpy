About crimeSpy

About version 1.0, last updated 20/09/15 by Ben Moskovitz

 

So what is crimeSpy?

crimeSpy is an open-source crime analysis software suite designed for use with data released by the Chicago Local Government, but is designed to be easily modifiable for use with other data. Using crimeSpy, you can plot individual crime datapoints on a map, look at them in a table view, and perform basic data analysis.

Where can I find data to use with crimeSpy?

As mentioned earlier, crimeSpy is designed for use with data released by the Chicago Local Government, which can be found here.

Alright, Ive got my data sorted out. How do I actually use crimeSpy?

Easy peasy! First of all, click on File -> Import, and navigate to your downloads folder and select the file youve just downloaded. Wait a second while it imports. Once its fully imported, click on the Crime Collection dropdown and select the path to the file you just imported. Your data should display, and then youre away laughing! You can import as many CSVs as you like and switch between them on the fly.
